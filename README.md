                                    01

In JavaScript, variables are declared using the keywords var, let, and const.

Variable: Function-scoped; can be redeclared and reassigned.

let: Block-scoped; cannot be redeclared but may be reassigned.

Const: Block-scoped, cannot be redeclared or reassigned, and must be initialized upon declaration.


                                     02

The spread operator (...) in JavaScript is used to expand elements of an array or properties of an object into individual elements.

                                     03

map(), filter(), and forEach() are array methods in JavaScript, but they behave differently:

map() → Creates a new array by transforming each element.

filter() → Creates a new array with elements that pass a condition.

forEach() → Loops through the array but does not return a new array.

                                     04

An arrow function is a shorter way to write a function in JavaScript using the => syntax.

like : const add = (a, b) => a + b;

                                      05

Template literals are a way to create strings in JavaScript using backticks ( ) instead of quotes.

They allow embedding variables and expressions using ${ }.

Example: let name = "Sam";
         console.log(`Hello ${name}`);
         Output: Hello Sam
